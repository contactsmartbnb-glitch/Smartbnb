# Base de connaissance micronutrition

Déposez ici les cours de micronutrition (un fichier par sujet) au format
`.txt` ou `.md`.

Ces fichiers sont automatiquement chargés par la fonction `analyse.js` et
servent de **base de connaissance** à l'IA : toutes les recommandations
s'appuient strictement sur leur contenu.

## Conseils d'organisation

- Un fichier par thème, ex. `perimenopause.md`, `thyroide.md`,
  `resistance-insuline.md`, `vitamine-d.md`…
- Texte clair, structuré en sections (le format n'a pas besoin d'être parfait).
- Évitez d'y mettre des données personnelles de patientes.

> Ce fichier d'instructions n'est pas chargé comme cours (il commence par `_`).
