# Pilier 5 — Immunité & inflammation

## En 30 secondes
Votre système immunitaire est votre armée intérieure. L'inflammation aiguë est un « feu bénéfique » qui détruit les ennemis. Mais quand ce feu ne s'éteint pas, vous entrez dans une inflammation chronique silencieuse — le terreau des maladies modernes. La micronutrition rééquilibre cette inflammation en renforçant votre terrain.

## Comprendre (grand public)

### Rôle dans le corps
L'immunité fonctionne en trois étages : les **barrières** (peau, muqueuses) forment la première ligne ; l'**immunité innée** réagit immédiatement et sans finesse ; l'**immunité adaptative** reconnaît précisément et se souvient.

L'inflammation aiguë est physiologique : vasodilatation, recrutement des globules blancs, élimination de l'envahisseur — puis **résolution active**. Aujourd'hui, l'inflammation s'initie correctement mais ne se résout pas. Elle devient **chronique à bas bruit**, silencieuse : pas de rougeur visible, juste une fatigue, des douleurs, une susceptibilité infectieuse.

### Pourquoi c'est central
- Toutes les maladies chroniques modernes (diabète, cardiovasculaire, neurodégénérescence, cancer, auto-immunité) reposent sur une inflammation chronique non résolue.
- L'équilibre des acides gras membranaires détermine si l'inflammation s'éteint ou s'auto-amplifie.
- Le microbiote intestinal dysbiosé → endotoxémie métabolique → inflammation systémique persistante.

### Quand ça dysfonctionne — les signes
- Infections à répétition malgré une hygiène normale
- Douleurs chroniques diffuses (arthralgies, myalgies)
- Allergies, eczéma, rhinites
- Fatigue persistante, brouillard mental
- Intolérances alimentaires
- Antécédents inflammatoires (fausses couches, thromboses)

## Données techniques (pour l'IA)

### Concepts clés
- **Innée vs adaptative** : innée = rapide, non spécifique (PAMPs/DAMPs → TLR → NF-κB) ; adaptative = lente, mémorisée (Ag → B/T clonale).
- **Lymphocytes T** : Th1 (IFN-γ), Th2 (IL-4), Th17 (IL-17, auto-immunité), Treg (IL-10, TGF-β, résolution).
- **Réponse biphasique** : offensive (IL-1, IL-6, TNF-α, PGE2) → résolution active (résolvines E/D, lipoxines) — carence en EPA/DHA bloque cette résolution.
- **Résolvines** : SPM (Specialized Pro-resolving Mediators) séries E (EPA) et D (DHA).
- **Ratio AA/EPA** : actuel 20-50:1 vs optimal 3:1.

### Mécanismes
- **Activation innée** : PAMPs (LPS, peptidoglycanes, ARN viral) → TLR (surtout TLR4) → kinases (IKK) → NF-κB libéré → cytokines.
- **Inflammation chronique silencieuse** : endotoxémie métabolique bas grade (LPS via leaky gut) → TLR4 hépatique → résistance insulinique.
- **Stress oxydant immunitaire** : burst oxydatif (myéloperoxydase, NOX) ; productif si pathogène, délétère sans antioxydants.
- **Tempête cytokinique** : hyperactivation → cytokines massives → choc septique.
- **Lien microbiote-immunité** : dysbiose → leaky gut → LPS systémique → inflammation persistante.

### Biomarqueurs et analyses pertinentes
- **CRP ultra-sensible (hs-CRP)** : < 1 mg/L optimal ; > 3 mg/L = inflammation chronique (meilleur prédicteur CV que LDL).
- **Ferritine** : aussi reflet du stress oxydant inflammatoire ; optimal 30-100 ng/mL (H), 15-60 (F).
- **NLR** (Neutrophil-Lymphocyte Ratio) : > 3 = inflammation systémique.
- **Vitamine D** : optimal ≥ 60 ng/mL pour immunité ; carence répandue (>80 % en France l'hiver).
- **Cytokines sériques** : IL-6, TNF-α, IL-10, IL-17 (coûteux).
- **Ratio AA/EPA** : > 10/1 = forte prédisposition inflammatoire.
- **Homocystéine** : > 15 µmol/L suggère carence B9/B12.

### Interventions micronutritionnelles
- **Vitamine D3** : 2 000-4 000 UI/j (maintien), 5 000-10 000 (correction) ; régule 200+ gènes immunitaires.
- **Oméga-3 (EPA/DHA)** : 1-2 g/j ; restaure la résolution.
- **Zinc** : 15-25 mg/j ; cofacteur immunitaire universel.
- **Vit A / β-carotène** : 3 000-5 000 UI/j ; synergie avec vit D.
- **Curcumine** : 500-1 000 mg/j ; inhibiteur NF-κB et COX.
- **Polyphénols** : quercétine (500 mg), EGCG, resvératrol.
- **Magnésium** : 500-1 000 mg élément/j.
- **Glutathion / NAC** : 500-1 500 mg/j.
- **Probiotiques** : L. plantarum, B. longum.

### Profils typiques / déclencheurs
- **Hyperactif** : infections fréquentes, fièvre élevée, eczéma poussées → régulation Treg, curcumine, vit D.
- **Inflammation chronique** : douleurs diffuses, allergies, intolérances, CRP > 3 → résolution (oméga-3, curcumine, antioxydants).
- **Dysbiose-endotoxémie** : fatigue, intolérances multiples, douleurs abdominales → scellement intestinal (L-glutamine, zinc, probiotiques).

## Pour le quizz « Où en es-tu ? »

### Symptômes à questionner (15 items)
1. Infections > 2-3/an
2. Allergies saisonnières ou alimentaires
3. Eczéma, psoriasis ou démangeaisons chroniques
4. Douleurs articulaires ou musculaires diffuses
5. Fatigue systématique après une infection
6. Problèmes digestifs récurrents (ballonnements, transit)
7. Aphtes ou herpès récidivants
8. Migraines ou céphalées fréquentes
9. Irritabilité, difficultés de concentration
10. Gonflements (articulations, paupières) sans cause mécanique
11. Antécédent de fausses couches ou thromboses
12. Prise régulière d'anti-inflammatoires ou cortisone
13. Adiposité abdominale élevée
14. Alimentation pauvre en poisson gras, riche en huiles ω6
15. Stress chronique sans récupération

### Profils de résultat (4)
- **Profil A — Immunité fragile / infections récidivantes** (items 1, 2, 5, 7, 9) : renforcer barrières (Zn, vit A), Th1 (vit D, fer), restaurer microbiote. Sommeil 8 h, douche froide progressive, complémentation Zn + D3 + A.
- **Profil B — Inflammation chronique active** (items 3, 4, 6, 8, 12) : inflammation non résolue → besoin résolvines. Oméga-3 EPA/DHA, curcumine, polyphénols, ↓ ratio AA/EPA.
- **Profil C — Dysbiose-endotoxémie métabolique** (items 6, 9, 13, 14) : leaky gut probable → endotoxémie → syndrome métabolique. Scellement intestinal (L-glutamine, zinc), réenzymation digestive, élimination aliments pro-perméabilité.
- **Profil D — Équilibre immunologique fragile multi-troubles** : profil mixte. Polymorphisme génétique probable (MTHFR, VDR). Bilan micronutritionnel complet, optimisation épigénétique.
