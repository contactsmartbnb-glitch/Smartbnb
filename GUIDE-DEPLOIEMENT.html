SmartBnBGuide déploiement v2

Mettre SmartBnB en ligne

4 phases · ~55 minutes · coût total : ~0–5 €/mois chatbot inclus

1
Google Sheets — base de données des leads
Chaque formulaire = une ligne + un email de notification
15 min
1
Va sur sheets.google.com → crée une nouvelle feuille → nomme-la SmartBnB CRM. Laisse-la vide, les colonnes se créent automatiquement.
2
Va sur script.google.com → Nouveau projet → nomme-le SmartBnB Leads
3
Supprime tout le code existant. Ouvre le fichier js/google-script.js inclus dans ton ZIP → copie-colle tout son contenu dans l'éditeur.
4
Modifie la ligne 8 avec ton vrai email :
// Remplace par ton email : const EMAIL_NOTIF = 'TON_EMAIL@gmail.com';
5
Dans le menu de gauche → Services (icône +) → ajoute Google Sheets API et Gmail API
6
Clique Déployer (haut droite) → Nouveau déploiement
Type : Application web · Exécuter en tant que : Moi · Accès : Tout le monde → Déployer
7
Google te demande d'autoriser → accepte tout → copie l'URL de déploiement (commence par https://script.google.com/macros/s/...)
Garde cette URL — tu en auras besoin à la phase 3.
8
Test : dans l'éditeur, sélectionne la fonction test → ▶ Exécuter → vérifie qu'une ligne apparaît dans le Sheet et que tu reçois un email.
2
Cloudflare Pages — hébergement du site
smartbnb.ma hébergé gratuitement · CDN mondial · HTTPS automatique
15 min
1
Va sur pages.cloudflare.com → crée un compte gratuit
2
Create a project → Direct Upload → nomme le projet smartbnb
3
Glisse-dépose le fichier smartbnb-complet.zip → Deploy site → 30 secondes et c'est en ligne sur smartbnb.pages.dev
Test : ouvre smartbnb.pages.dev et smartbnb.pages.dev/crm.html — tout doit s'afficher.

Connecter ton domaine smartbnb.ma

4
Dans Cloudflare Pages → ton projet → onglet Custom domains → Set up a custom domain → tape smartbnb.ma
5
Cloudflare t'affiche les enregistrements DNS à ajouter. Connecte-toi à ton registrar (là où tu as acheté smartbnb.ma) et ajoute :
Type Nom Valeur CNAME smartbnb.ma smartbnb.pages.dev CNAME www smartbnb.pages.dev
Délai DNS : ça peut prendre 5 minutes à 24h. C'est normal, pas de panique.
6
Une fois propagé → retourne dans Cloudflare → Check DNS records → si tout est vert, smartbnb.ma est connecté ✓
3
Configurer le CRM + connecter Google Sheets
2 minutes · une seule fois
5 min
1
Ouvre smartbnb.ma/crm.html → connecte-toi avec le mot de passe : smartbnb2026
2
Clique Configuration → colle l'URL Google Apps Script de la phase 1 → Enregistrer
3
Change immédiatement le mot de passe dans la section "Mot de passe admin" → choisis quelque chose de solide.
Ne laisse pas le mot de passe par défaut smartbnb2026. Le CRM est accessible à quiconque connaît l'URL.
4
Test final : remplis un formulaire sur le site → vérifie ① une ligne dans Google Sheets ② un email reçu ③ le lead dans le CRM
4
Chatbot Claude — Cloudflare Worker
Le bot "Sami" branché sur Claude Haiku 4.5 · ~0–5 €/mois selon le trafic
10 min

A · Récupérer une clef API Anthropic

1
Va sur console.anthropic.com → crée un compte → Settings → API Keys → Create key → nomme-la smartbnb-prod.
2
Copie la clef (commence par sk-ant-...) et garde-la en lieu sûr — elle ne s'affichera plus jamais.
Crédite ~5 € sur ton compte Anthropic (Billing → Add credits). Avec Haiku 4.5 + prompt caching, ça paie ~5 000 conversations.

B · Déployer le Worker Cloudflare

3
Sur dash.cloudflare.com → Workers & Pages → Create → Create Worker → nomme-le chat-smartbnb → Deploy (laisse le code par défaut pour l'instant).
4
Une fois créé → Edit code → supprime tout → copie-colle l'intégralité de js/worker.js → Save and deploy.
5
Onglet Settings → Variables and Secrets → Add :
Type : Secret Name : ANTHROPIC_API_KEY Value : sk-ant-... (ta clef de l'étape 2)
Puis Save et redéploie le Worker.
Choisis bien "Secret" et pas "Text". Un Text serait visible en clair par toute personne ayant accès au tableau de bord.
6
Copie l'URL du Worker (Settings → Triggers, ou tout en haut de la page Worker). Format : https://chat-smartbnb.TON-PSEUDO.workers.dev

C · Brancher le widget sur le Worker

7
Ouvre js/chatbot.js → repère la ligne const CHAT_API_URL = ... tout en haut → remplace l'URL placeholder par celle de ton Worker.
// AVANT : const CHAT_API_URL = 'https://chat-smartbnb.TONPSEUDO.workers.dev'; // APRÈS (exemple) : const CHAT_API_URL = 'https://chat-smartbnb.youness-akhalouf.workers.dev';
8
Si tu as déjà déployé en phase 2 → re-upload le ZIP sur Cloudflare Pages pour pousser la nouvelle version. Sinon, fais-le après l'étape phase 2.

D · Tester le chatbot

9
Ouvre smartbnb.ma → clique sur le bouton chat en bas à droite → pose une question type "Quel rendement à Marrakech ?". La réponse doit être variée et personnalisée (Claude), pas toujours la même phrase (fallback).
Comment savoir si Claude répond vraiment ? Pose 2 questions très différentes — si les réponses sont vivantes et naturelles, c'est Claude. Si c'est toujours la même réponse pour le même mot-clef, c'est le fallback hors-ligne (donc le Worker n'est pas joignable).
10
Surveille la facture : dans console.anthropic.com → Usage, tu vois la conso en temps réel. Haiku 4.5 + prompt caching : ~0,0005 € par conversation. 5 € de crédit = plusieurs milliers d'échanges.
